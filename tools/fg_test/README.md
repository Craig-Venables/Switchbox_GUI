# Function Generator config test

Standalone diagnostic for the Siglent SDG1032X. Verifies that settings sent by the Laser FG Scope GUI are accepted and read back correctly.

## Run

```powershell
python tools/fg_test/test_fg_config.py
```

Paste the full terminal output when reporting FG configuration issues.
