# Motor Control GUI - Changelog

## Recent Improvements (Dec 2025)

### Layout Enhancement - Collapsible Controls

**Changes:**
- Added collapsible sections for all control groups (Jog, Go-To, Motor Settings, Presets, Scan, Function Generator, Laser)
- Implemented "Expand All" and "Collapse All" buttons for quick navigation
- Increased window size from 1200x750 to 1400x900 for better visibility
- Improved control panel width from 320px to 380px
- Reduced padding between sections (from 5px to 3px) for more compact layout
- Set advanced sections (FG, Laser) to start collapsed by default

**Benefits:**
- Easier to navigate - focus on what you need
- Better screen space utilization
- Cleaner, more organized interface
- All controls remain accessible without scrolling excessively

**How to Use:**
- Click any section header or arrow (▼/▶) to expand/collapse that section
- Use "Expand All" button to see all controls at once
- Use "Collapse All" button to maximize canvas view
- Frequently used controls (Jog, Go-To, Presets) start expanded
