# Oscilloscope Pulse GUI

A simplified GUI for pulse capture with an SMU and oscilloscope.

## Purpose

- Send a pulse from the SMU
- Read raw waveform data from the scope
- Save raw data with calculated current and resistance

## Main UI Tabs

- `Measurements` - pulse/sweep settings, plot, run/save actions
- `Connections` - system/addresses, connect status, setup note
- `Help` - quick in-app help and guide shortcut

## Data Output

Saved columns:

- `Time(s)`
- `V_shunt_raw(V)`
- `Current(A)`
- `Resistance(Ohm)`
