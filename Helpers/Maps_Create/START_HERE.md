**Quick Start Guide for Non-Programmers**

## Step 1: Launch the Tool

Double-click this file or run in terminal:
```bash
python device_map_tool.py
```

## Step 2: Select Your Image

1. Click **"Browse..."** next to "Image File"
2. Find and select your device image
3. Click **"Open"**

## Step 3: Configure

1. **Sample Type:** Enter a name (e.g., "Cross_bar", "My_Sample")
2. **Section:** Enter a section name (e.g., "A", "B", "1")

## Step 4: Load Image

1. Click **"Load Image"**
2. A new window will open with your image
3. Read the instructions that appear

## Step 5: Map Devices

**Important:** Always drag from **top-left** to **bottom-right**!

1. Position mouse at **top-left corner** of a device
2. **Click and hold** left mouse button
3. **Drag** to **bottom-right corner**
4. **Release** mouse button
5. A green rectangle appears = success!

Repeat for each device.

## Step 6: Check Your Work

Click **"Check Mapping"** to verify everything is correct.

## Step 7: Save

Click **"Save Mapping"** when finished!

## Keyboard Shortcuts (in image window)

- **S** = Save current progress
- **Q** = Quit (auto-saves)
- **R** = Reset current device
- **U** = Undo last device
